AutoWaves can be run like any windows program. However before first-time operation it must be run in set-up mode. This allows the user to generate and export the required text parameters file. To run the set up utility, the user must double click on the program executable. In Windows, AutoWaves is installed in Program Files by default. The AutoWaves setup GUI will launch with the following splash screen:

 

​

 

The setup GUI allows users to select AutoWaves processing options and export the required run_parameters text file. First-time users may want to consult the AutoWaves user’s guide that is available from the Help button on the setup GUI.
