The modular design of XWaves allows a significant amount of flexibility to switch between data analysis tools and visualization tools. A typical analysis session might proceed as follows:

 

1. Select raw data for display and processing from the Start Up screen

 

2. Set data import characteristics in the Import Parameters Table

 

3. Use Data View Module to explore data prior to processing.

 

4. Subsample, Interpolate, or smooth input data using Utilities.

 

5. Select Wave Systems Module to isolate and track wave system evolution.

 

6. Set processing choices in Wave Systems Parameters Table.

 

7. Explore results in Wave Systems wave vector history display.

 

8. Navigate back to Data View for a variety of wave component displays

 

9. Select Swell Map to view estimated swell source locations and propagation routes.

 

10. Use Spectral Fit Module to compute fit parameters

 

11. Use Sea State Module to generate a sea state history and scatter tables

 

12. Explore results in Data View.
