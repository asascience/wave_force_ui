Description

Standard directional wave spectrum files (*.spt) output from the Datawell MkII Waverider buoys. These ASCII-formatted files contain a single spectrum per file. Date and time information is obtained from the filename, for example 09231400.SPT (MMDDhhmm.spt). The software can read other date/time combinations in Datawell .spt filenames; for example YYYY_DD_MM_hh_mm.spt would also be allowed.

Format

File Header (12 parameters)

tn

transmission index (1 to 8)

Hs

significant wave height [cm]

Tz

zero-upcross period [s]

Smax

maximum of the psd S(f) [m^2/Hz]

Tref

reference temperature [°C, centigrade]

Tsea

Sea surface temperature [°C, centigrade]

Bat

Battery status (0 = empty to 7 = full)

Av

offset of the vertical accelerometer

Ax

offset of the x-accelerometer

Ay

offset of the y-accelerometer

Ori

buoy orientation [°]

Incli

magnetic inclination [°]

 

Spectrum data (64 x 6 array)

f, S(f) / Smax, Dir(f), Spr(f), Skew(f), Kurt(f)

 

f

wave frequency [Hz]

S(f) / Smax

relative psd (power spectral density) [-]

Dir(f)

wave direction [°]

Spr(f)

directional spread [°]

Skew(f)

skewness of the directional distribution [-]

Kurt(f)

kurtosis of the directional distribution [-]

 

Additional information is available from Datawell on their Data Format Specifications page.
