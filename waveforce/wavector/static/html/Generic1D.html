Description

Generic 1D wave spectra file in binary MATLAB .mat file format.

 

 

Format

td

Nx5 array containing date time information with columns 1-5 being YYYY, MM, DD, HH, MM respectively. *Can also have a 6th column for seconds

fd

1xM array containing all of the frequency bins (Hz)

spec1d

NxM matrix containing 1D wave spectra (m^2/Hz)

 

where N is the number of records and M is the number of frequency bins
