AutoWaves program options are specified in the Parameters Text File. This typically has the filename run_parameters.txt, however the user can select any filename desired (with a .txt extension). The critical contents of this file are reproduced below. Descriptions follow each set of options (in red). Normally there will be no need for the user to open or edit the text parameters file, as these functions are now controlled by the set-up utility.
