Description:

The Wave Profiler allows the user to measure non-directional wave parameters continuously for long periods of time with high temporal and spatial resolution in deeper water from the safety of an underwater mooring.  The instrument is particularly useful for clients who need to measure long-period waves (such as rogue waves or infra-gravity waves), waves in wash and in wakes, and non-linear waves. Data is output in 2 ASCII text files: a file containing the raw spectra time series (.dat), and a header file containing metadata and frequency bins. The files must have the same name but different file extensions. For processing multiple files there must be a corresponding .hdr file for each raw data file and the files must contain the same frequency bins.

Format

Header file

There are 13 lines of header data:

$file_name

Name of data file

$file_type

File type (ascii)

$start_date

Date and time of startup in UTC (YYYY MM DD HH MM SS.SSS)

…

…

 

Data file

The data file consists of the 1 dimensional wave spectra, one line per time record with the following columns:

Time_offset_from_start_date(s) burst_period(s) S(fn) … S(fnfreq)

 

There are no headers in the data file and the data are tab delimited.

 

The time column is the time in seconds after the start date ($start_date 2013 1 13 14 59 59.998 UTC). The time refers to when the wave bursts (4096 s) begin. Thus, half of the burst period is added to the start time to prevent an artificial phase shift.
