Swell Map Module

Swell Map Legend The Swell Map Legend can display up to 27 Events at a time and provides access to settings for each Event.

Event Menu

These options are also available in the Events menu in the Options pull down menu or by right clicking on the top legend buttons as described above.

Event Information       Display available Event information in a pop-up dialog window.

Zoom to Event             Automatically adjusts map to view Event.  Event will be set visible if currently invisible.

Show Event Only         Set Event visible and all other Events will be set to invisible.

Edit Event                     Submenu containing the following options for adjusting settings for each Event.

 

     Event Color Adjust color of Event track line and marker.

     Marker Size Adjust size of Event marker color.

     Line Style Adjust line style of Event track line.  Solid line is default.

     Line Width Adjust line width of Event track line.  1 is default.
