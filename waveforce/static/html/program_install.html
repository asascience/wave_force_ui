XWaves is designed to run in the MS Windows, Macintosh and Linux O/S environments. To  optimize  menu  legibility  Windows  XP  users  should  switch  control  panel  display settings to “classic view”.

 

To install from a CD follow these simple steps:

1. Decide where to install XWaves on your computer hard disk.  The program will operate from any location that you have read/
    write access to.

2. Drag or copy the XWaves folder from CD-ROM to the selected location on your PC hard disk

3. Ensure that the XWaves folders, and all subfolders, are not write-protected by right-clicking on the XWaves folder and selecting
    Properties.  The Read Only box should not be checked, and this condition should be applied to all subfolders and files.

4. Start MATLAB, and change the MATLAB Current Folder to the XWaves folder.  This step only needs to be done the first time
    you run XWaves.

5. Type xwaves at the command prompt and follow the installation instructions.

 

For reading and saving data, XWaves provides the ability to browse to any folder of choice.  See Data Formats for an overview of XWaves data structures.
