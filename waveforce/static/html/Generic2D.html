Description

Generic 2D directional wave spectra file in binary MATLAB .mat file format.

 

Format

td

Nx5 array containing date time information with columns 1-5

being YYYY, MM, DD, HH, MM respectively.

*Can also have a 6

th

 column for seconds

fd

1xNFREQ array containing all of the frequency bins (Hz)

thetad

1xNANG array of direction bins (deg)

spec2d

NxNFREQxNANG matrix containing 2D directional wave spect

ra (m^2/(Hz*radians))

 

where N is the number of records, NFREQ is the number of frequency bins, and NANG is the number of direction bins
