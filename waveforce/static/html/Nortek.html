Description

The Nortek *.wft data files are ascii-formatted data exported from the Nortek STORM software. Storm is Nortek's graphical, wave and current profile data processing software. It accepts raw data files from the Nortek series of Aquadopp Profilers and the AWAC.   The *.wft file export option was created by Nortek specifically for WaveForce Technologies software products.  There can be multiple records per file.
