Format

File Header

time: hh:mm:ss DD MMM YYYY

nfreq: Number of frequency bins

ndir: Number of direction bins

freq(1): First frequency bin value (Hz)

freq_width: Frequency bin spacing (Hz)

dir_width: Direction bin spacing (Deg)

2 additional header lines

 

Spectrum data

All values for dir1(1..nfreq) are given, then all values for dir2(1..nfreq), and so on (1..ndir).
