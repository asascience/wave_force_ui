There are a number of available data export options from the Data View Module

 

The Tables pull-down menu provides the following options for displaying and exporting tables:

 

Export Wizard – Custom export of data.  See Data Export Wizard for help.

Fixed Format – For exporting a variety of fixed-format ASCII statistics tables

Sea State – A variety of options for exporting results from the Sea State Module:

Sea State History Table – a full listing of the computed Sea State parameters.

Joint Occurrence Table – A Joint Occurrence Table computed from the Sea State History Table contents

Scatter Tables – A series of Scatter Tables computed from the Sea State History Table contents

Extreme Analysis – Peaks table and recurrence interval tables

 

For each Sea Table the user can select from the following output options:

 

View Table – A separate text window opens with table contents

Export to ASCII file – An ASCII text file with table contents is saved to disk

Export to Microsoft Excel file – A Microsoft Excel file with table contents is saved to disk
