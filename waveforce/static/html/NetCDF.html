Description:

XWaves includes an interface to read netCDF (Network Common Data Form) file containing wave spectra data.

 

Example

The netCDF files follow the netCDF standard. See below for a list of variables needed for XWaves processing.

 

Format

Variables  include:

Name Data set name

Serial Dates Observation Date and Time as MATLAB Serial Date Number

Station Longitude Longitude (decimal degrees)

Station Latitude Latitude (decimal degrees)

Station Water Depth Water depth (m)

Frequencies Frequency bins (Hz)

Sectors Direction bins (degrees)

2D Directional Spectrum Directional Wave Spectra (m2/(hz x deg)

Wind Direction Wind Direction (Deg From North)

Wind Speed Wind Speed (m/s)

Wind Data Type

Frequency Band Widths Array of Frequency Bin Differentials (Frequencies(n+1) â€“ Frequencies(n))

 

The variable names do not have to match those above. XWaves has a user interface to allow the user to match the appropriate variables in the netCDF file to necessary variables above.

 

Additional information on the netCDF format is available from UCAR.
