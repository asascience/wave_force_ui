Format:

Data file: 2 header lines then data lines.

Data line has date, time, compass TS, heave TS, pitch TS, roll TS.

One collection per line.

TimeN73.5_E15.5, N73.5_E15.5, N73.5_E15.5, ...     (Line 1)

Timecompass[1],  compass[2],  compass[3], ...      (Line 2)
